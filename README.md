# Hacker News Scraper Pro

Scrape Hacker News stories, jobs, Show HN posts, Ask HN, comments, and full-text search results.

## Features

- 6 story categories: top, new, best, ask, show, jobs (Who's Hiring)
- Full-text search via Algolia
- Optional comment extraction (top-level comments per story)
- Returns: title, URL, points, comment count, author, timestamp
- Up to 500 items per run
- No login required — uses official HN Firebase API

## Use Cases

- Track tech trends and trending startups
- Monitor "Who's Hiring" for job market intel
- Watch Show HN for new product launches
- Build news aggregators or alert systems
- Sentiment analysis on tech topics

## Output

JSON array with story metadata. Use Apify's CSV/Excel export for analytics.
